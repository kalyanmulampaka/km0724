# Getting Started

# Build
1. Execute the following command to build the service

    `mvn clean install`

# Run
1. Execute the following command to run the service

    `mvn spring-boot:run`

# API Documentation
REST API documentation is available at: http://localhost:8080/swagger-ui/index.html
